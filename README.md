# RayTracer in Python
A Python Implementation of Ray Tracing in One Weekend.

The cpp version is in this [repository](https://github.com/YanjieZe/RayTracing).

# Usage
```
python main.py --img_width 256 --sample 1 --recursion 20 --thread 16
```
